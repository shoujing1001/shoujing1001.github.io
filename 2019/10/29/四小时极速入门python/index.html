pyhton中的数据类型

学习任何一门语言的第一步就是先了解这门语言中有那些数据类型，而后才好根据不同数据类型对数据进行正确操作。我以数据复杂程度将python中数据类型分为基础数据类型，简单数据类型为:整数、浮点数、字符串、布尔值、和空值(None)。接下来分基础和高级数据类型分别总结:

基础数据类型

整数&&浮点数

整数和浮点数与数学中的定义大致相同，值得注意的是python中浮点数的运算是不精确的，而整数的运算则是精确的。另外整数和整数计算的结果总是整数，浮点数与浮点数的计算结果总是浮点数，但是整数与浮点数的运算结果则为浮点数。基于此衍生出整数在做除法时，即使除不尽结果也仍为整数，余数被忽略了:

字符串

python中字符串规则与js中类似，单引号或双引号以及相互包含用以表示字符粗引号的规则都是一样的，注意一下转义字符\的使用。
关于单行字符串r'...'和多行字符串'''...'''有一个注意点为，r'...'中不能有单引号或双引号，而'''...'''则可以自由使用。

布尔值

true or false，没太多需要解释的，布尔值每种语言中条件判断所必须的数据类型，布尔值有几种或、与、非运算，运算规则与高中数学知识大同小异。布尔值的运算需要注意短路运算的存在，这里不注意时可能被坑到:

  1. 在计算 a and b 时，如果 a 是 False，则根据与运算法则，整个结果必定为 False，因此返回 a；如果 a 是 True，则整个计算结果必定取决与 b，因此返回 b。
  2. 在计算 a or b 时，如果 a 是 True，则根据或运算法则，整个计算结果必定为 True，因此返回 a；如果 a 是 False，则整个计算结果必定取决于 b，因此返回 b。

空值(None)

空值是一个特殊概念，不能简单理解为0，0至少表示了数量0的概念，None表示的则是一种类似道家空的概念。pyhton中如果函数没有返回任何东西时输出则会看到None。

参数

函数处理一些事件时，往往需要一些参数以实现同一个函数能灵活处理很多同类型事件，比如我们前面定义的add(num1,num2)就是处理两数相加的函数，其后的num1、num2就是这个函数的参数，参数的存在使得函数拥有了更多的灵活性。

Python的函数定义非常简单，但灵活度却非常大。除了正常定义的必选参数外，还可以使用默认参数、可变参数和关键字参数，使得函数定义出来的接口，不但能处理复杂的参数，还可以简化调用者的代码。

add(num1,num2)这个函数中的num1 num2就是两个位置参数，调用函数时，传入的两个值按照位置顺序依次赋给参数num1和num2。

在使用默认参数定义函数时，需要注意以下问题：
1. 默认参数应该在必选参数之后定义，否则会报错
2. 当函数有多个参数时，把变化大的参数放前面，变化小的参数放后面。变化小的参数就可以作为默认参数
3. 定义默认参数要牢记一点：默认参数必须指向不变对象
4. 有多个默认参数时，如果只需要自定义其中某一个参数，可以使用参数名=参数值的方式向函数传入这个自定义的默认参数

切片

我们在了解list和tuple的时候使用具体的索引或for循环取其中的一个或部分元素，其实python为我们提供了更为方便的工具——切片。我们先来简单看一下它的用法：

上面的L[0:3]，表示的是从索引0开始取，直到索引3为止，但不包括索引3。即索引0，1，2，正好是3个元素。如果第一个索引是0，还可以省略：L[:3]。

切片操作还有第三个参数，表示每N个取一个。如：L[0:3:2]取出的结果是['Michael', 'Tracy']。

切片操作还支持倒序，L[-1]表示取最后一个元素。倒序切片包含起始索引，不包含结束索引。如L[-4:-1:2]的结果为['Sarah', 'Bob']。表示从倒数第4个开始到倒数第一中每2个取出一个元素的结果。

另外，字符串也可以看作特殊的list进行切片操作，规则和list一致。例如字符串"ABCDEFG"[1:6:2]的结果为'BDF'。
